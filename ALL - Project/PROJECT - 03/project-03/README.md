## Project Name: BookStock

BookStock is a simple web-based library management system built with Laravel.

## Features

- Add New Book
- View Book List
- Edit Book
- Delete Book
- Upload Book Cover Image
- Category & Author Management (Relational Database)
- Form Validation
- File Upload with Storage System

## Technology Used

- Laravel
- PHP
- MySQL
- Bootstrap / Tailwind
- Query Builder (DB Facade)
