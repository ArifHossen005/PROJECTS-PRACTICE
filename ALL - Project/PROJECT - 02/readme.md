## Project Overview

This is a **User Authentication and Profile Management System** built using  **PHP (OOP), MySQL, and Tailwind CSS** .

Users can register, log in, view and update their profile, and log out securely.

---

## Features

* **User Registration:** Name, email, and password (hashed). Email must be unique.
* **User Login:** Email and password verification. Maintains session for logged-in users.
* **Profile Management:** View and update profile (name, email, password). Only the logged-in user can update their profile.
* **Logout:** Securely destroys session and redirects to login.
